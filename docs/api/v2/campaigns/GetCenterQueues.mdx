---
title: "List contact-center queues"
description: "Return all active queues (voice and chat) for the authenticated account."
api: "POST https://<base>/api/v2/GetCenterQueues.php"
---

Returns every active row in `cc_fila` for the authenticated customer (`ccf_active = 1`), sorted alphabetically by name. Useful to populate queue pickers in custom dashboards or before calling [`CreateCampaign`](/docs/api/v2/campaigns/CreateCampaign) with `queue_id`.

## Authentication

Token authentication required. See [Authentication](/docs/api/v2/authentication).

## Request

<ParamField body="tsid" type="string" required>
  Token for the account.
</ParamField>

This endpoint takes no other parameters.

## Request example

<RequestExample>

```bash curl
curl -X POST 'https://<base>/api/v2/GetCenterQueues.php' \
  -H 'Content-Type: application/json' \
  -d '{ "tsid": "YOUR_TSID" }'
```

</RequestExample>
